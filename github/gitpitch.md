# GitPitch

* `PITCHME.md` 파일로 온라인 상에서 프레젠테이션을 할 수 있습니다.
* https://gitpitch.com/[user]/[repo]/[branch] 주소로 접근하여 프레젠테이션을 볼 수 있습니다.
* `Offline` 을 클릭하여 오프라인에서도 볼 수 있는 파일을 받을 수 있습니다.

## `PITCHME.md` 파일 형식

* 마크다운 형식을 사용합니다.
* 슬라이드 구분은 `#HSLIDE` 와 `#VSLIDE` 로 합니다 (`#` 다음에 공백이 없어야 합니다).
* 끝입니다.

```md
#HSLIDE

#VSLIDE

```
